This README file describe the files included in the submission of the Enron Fraud Project:
- README.txt: This file
- Answer.html: The file with the answers of the "nron Submission Free-Response Questions" (The images directory is needed to see the graphs)
- poi_id.py: Python code to generate the pkl files for the project
- my_classifier.pkl: The pickle file which descrive the classifier
- my_dataset.pkl: The pickle file of the dataset
- my_feature_list.pkl: The pickle file of the selected features
- Reference.txt: A list of the reference used to do the project.
